# 사용자 가이드

## 한국코프론 스마트 유통 플랫폼

에어컨 냉매·가스·부품을 쉽고 빠르게 주문할 수 있는 B2B 플랫폼입니다.

## 시작하기

### 회원가입 및 로그인

1. **회원가입**
   - 우측 상단의 "로그인" 버튼 클릭
   - "회원가입으로 전환" 선택
   - 이메일과 비밀번호 입력
   - "회원가입" 버튼 클릭

2. **로그인**
   - 우측 상단의 "로그인" 버튼 클릭
   - 이메일과 비밀번호 입력
   - "로그인" 버튼 클릭

## 고객용 기능

### 상품 둘러보기

1. **홈페이지 방문**
   - 플랫폼의 주요 기능 소개 확인

2. **상품 목록 보기**
   - 상단 메뉴에서 "상품" 클릭
   - 카테고리별 상품 목록 확인
   - 상품 카드 클릭으로 상세 정보 확인

3. **상품 상세 정보**
   - 상품명, 설명, 가격, 재고 확인
   - "장바구니에 담기" 버튼으로 장바구니 추가

### 장바구니 관리

1. **장바구니 담기**
   - 상품 목록 또는 상세 페이지에서 "장바구니에 담기" 클릭

2. **장바구니 확인**
   - 상단 메뉴에서 "장바구니" 클릭
   - 담긴 상품 목록과 수량 확인

3. **수량 조절**
   - "+" 또는 "-" 버튼으로 수량 변경
   - 수량이 0이 되면 자동으로 상품 삭제

4. **장바구니 비우기**
   - "장바구니 비우기" 버튼으로 전체 삭제

### 주문하기

1. **주문 생성**
   - 장바구니에서 "주문하기" 버튼 클릭
   - 로그인하지 않은 경우 로그인 페이지로 이동

2. **주문 확인**
   - 주문 완료 시 주문번호 표시
   - 장바구니 자동 비우기

3. **주문 내역 확인**
   - 상단 메뉴에서 "주문내역" 클릭
   - 주문 상태 및 상세 정보 확인

### 주문 상태 설명

- **대기중**: 주문 접수됨, 관리자 승인 대기
- **승인됨**: 관리자가 주문을 승인함
- **배송중**: 상품이 배송 중
- **배송완료**: 배송 완료
- **취소됨**: 주문 취소됨

## 관리자 기능

### 대시보드

1. **접근 방법**
   - 관리자 계정으로 로그인
   - "/admin" 경로로 접근

2. **주요 지표 확인**
   - 총 주문 수량
   - 총 매출액
   - 재고 부족 상품 수
   - 신규 고객 수

### 주문 관리

1. **주문 목록 보기**
   - 대시보드에서 "주문관리" 메뉴 클릭
   - 모든 고객의 주문 목록 표시

2. **주문 상태 변경**
   - 각 주문의 상태 드롭다운에서 변경
   - 실시간으로 상태 업데이트

3. **주문 상세 정보**
   - 주문번호, 고객 정보, 상품 목록, 금액 확인

## 상품 관리 (향후 구현)

### 상품 등록
1. 관리자 메뉴에서 "상품관리" 선택
2. 상품 정보 입력 (이름, 카테고리, 가격, 재고 등)
3. 이미지 업로드
4. 등록 완료

### 상품 수정
1. 상품 목록에서 수정할 상품 선택
2. 정보 수정
3. 저장

## 재고 관리 (향후 구현)

### 재고 현황 확인
1. "재고관리" 메뉴 선택
2. 상품별 재고 수량 확인
3. 부족 상품 알림 확인

### 재고 조정
1. 상품 선택
2. 입고/출고 수량 입력
3. 재고 업데이트

## 가격 정책 관리 (향후 구현)

### 고객별 가격 설정
1. "가격정책" 메뉴 선택
2. 고객 선택
3. 상품별 할인율 또는 고정 가격 설정

## 알림 및 커뮤니케이션

### 자동 알림
- 주문 생성 시 관리자에게 이메일 알림
- 배송 상태 변경 시 고객에게 알림톡 발송
- 재고 부족 시 관리자에게 알림

### 수동 알림 (향후 구현)
- 고객에게 공지사항 발송
- 프로모션 알림

## 모바일 사용

### 반응형 디자인
- 모든 기능이 모바일에서 정상 작동
- 터치 친화적 인터페이스

### 모바일 앱 (향후 구현)
- React Native 기반 네이티브 앱
- 푸시 알림 지원

## 문제 해결

### 로그인 문제
- 이메일과 비밀번호 확인
- 인터넷 연결 상태 확인
- 브라우저 캐시 삭제 후 재시도

### 주문 오류
- 장바구니에 상품이 있는지 확인
- 로그인 상태 확인
- 네트워크 연결 확인

### 페이지 로딩 문제
- 브라우저 새로고침
- 캐시 삭제 (Ctrl+F5)
- 다른 브라우저에서 시도

## 자주 묻는 질문 (FAQ)

### Q: 상품 가격은 어떻게 결정되나요?
A: 기본 가격에 고객별 할인 정책이 적용됩니다. 자세한 사항은 영업팀에 문의하세요.

### Q: 배송은 얼마나 걸리나요?
A: 지역에 따라 1-3일 소요됩니다. 주문 시 배송 일정 확인 가능합니다.

### Q: 반품/교환 정책은 어떻게 되나요?
A: 상품 수령 후 7일 이내에 반품 가능합니다. 자세한 정책은 이용약관을 확인하세요.

### Q: 대량 구매 시 할인이 되나요?
A: 네, 수량에 따라 추가 할인이 적용됩니다. 영업팀에 문의하세요.

### Q: 재고 부족 상품은 언제 입고되나요?
A: 재고 상황에 따라 다르니, 상품 페이지에서 "문의하기"를 이용하세요.
